# Control para TV LG (webOS) - Gratis y sin anuncios

Esta es una aplicación sencilla para controlar televisores LG Smart TV que usen el sistema operativo webOS.

## Por qué creé esta aplicación

Desarrollé esta aplicación por necesidad propia, ya que el control físico de mi televisor se rompió y dejó de funcionar. Al buscar opciones en la tienda de aplicaciones de Android, me di cuenta de que todas las alternativas gratuitas estaban llenas de anuncios molestos que hacían incómodo algo tan simple como cambiar el volumen o apagar la tele.

Por eso decidí hacer mi propia versión. Es totalmente gratis, funciona bien y no tiene ningún tipo de publicidad.

## Características

* Descubrimiento automático: Busca y encuentra televisores LG en tu misma red local.
* Botones principales: Apagado, control de volumen, silenciar, subir y bajar canales.
* Teclado numérico: Teclado de canales para escribir directamente el canal al que deseas cambiar y enviarlo.
* Teclado en pantalla: Permite escribir texto en el televisor desde el celular de forma más cómoda.
* Lanzador de aplicaciones: Puedes abrir aplicaciones instaladas en la TV (como Netflix o YouTube) desde el teléfono.
* Respuesta táctil: Pequeñas vibraciones al presionar los botones para sentir que estás usando un control real.

## Compatibilidad

Funciona con cualquier televisor LG Smart TV con sistema webOS (modelos de 2014 en adelante).

Importante: Tanto el televisor como el celular tienen que estar conectados a la misma red Wi-Fi.

## Cómo instalar

1. Descarga el archivo LG_Remote_Control.apk que está en este repositorio.
2. Abre el archivo en tu celular e instálalo. Si tu teléfono lo solicita, deberás permitir la instalación de aplicaciones de fuentes externas.
